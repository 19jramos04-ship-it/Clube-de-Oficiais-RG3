Acesso Online & QR Code

Para gerar QR codes e ter o bar acessível a todos, o ficheiro precisa de estar publicado online.
A forma mais simples e gratuita é o GitHub Pages — sem custos, sem registo em servidores pagos.

Como publicar (uma vez)
  1. Crie conta gratuita em github.com
  2. Crie um repositório novo com o nome bar-clube
  3. Faça upload deste ficheiro HTML com o nome index.html
  4. Nas Settings do repositório → Pages → Source: main branch → Guardar
  5. O seu bar ficará disponível em: https://SEU-UTILIZADOR.github.io/bar-clube/
⚠ Nota importante sobre os dados
